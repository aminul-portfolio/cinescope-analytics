CineScope Analytics

🎬 About

CineScope Analytics is a Django portfolio project that demonstrates a modern movie discovery experience: fast search, smart filters, availability cues, and clean detail pages — packaged with a SaaS-style UI.

Our mission

Help viewers decide quickly what to watch. The app reduces time-to-choice using keyword search, faceted filters, and “Watch Available” visibility so users don’t waste time clicking dead ends.

Why this improves retention

  • ✅ Reduces “no result” frustration with broad matching (title/cast/description/genres)
  • ✅ Reduces decision fatigue with filters (year/genre/category/duration/sort)
  • ✅ Shows watchability instantly via Watch Available badges
  • ✅ Prioritizes quality with Top Rated ordering (rating + rating count)
  • ✅ Encourages exploration with related + recommended titles

Core product features

🔍 Search + Discovery
Keyword search across title, cast, description and many-to-many genres. Smart ordering prioritizes top-rated content when filtering.
✅ Availability Intelligence
Movies show Watch Available status so users avoid dead-end clicks. Optional “Watch Available only” filter.
⭐ Per-user Ratings
Each user can rate a movie once and update later. Aggregates update instantly across movie detail, list/cards, and analytics tables.
💬 Comments
SaaS-style comment form and community feed on each movie page (login required). Supports moderation via an is_public flag.
🏗️ Analytics ETL Pipeline
Built a repeatable data pipeline: raw events → DailyMetric aggregates → dashboard + CSV exports. Supports backfill + daily incremental runs via a management command.
🔐 Accounts
Signup/login/logout, dashboard access, and password reset flows.

Analytics (manager view)

A staff-only analytics dashboard supports product decisions and operational visibility via:

  • 📅 Date ranges: 7 / 30 / 90 days (trend + funnel + top tables)
  • 📈 Total views (overall interest)
  • ▶️ Watches & ❤️ Favorites (range-based)
  • 👤 Active users (7d vs selected range)
  • 🏆 Top movies / genres / categories
  • ⭐ Top rated with rating count confidence
  • 🩺 Data Health panel (missing poster/trailer, no watch links, unrated items)
  • ⬇️ CSV exports for top tables (enterprise reporting)
  • 🏗️ ETL-backed chart sourced from DailyMetric (gold layer)
